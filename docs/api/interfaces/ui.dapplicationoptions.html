Optional overlay

overlay: undefined | false | true
  • Defined in d-application.ts:61

An overlay layer setting. The overlay layer is used by modal UI elements including DDialog to provide the better interoperability with DOM-based UI elements and WebGL-based UI elements. However, an extra layer requires extra memory. Set to true to enable the overlay layer. The default value is false.
